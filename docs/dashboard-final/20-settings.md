## Settings Sections

### Organization Profile
- Logo upload
- Organization name
- Website URL
- Contact information

### Bank Accounts
- Add/manage bank accounts
- Set default account for payouts
- Verify account ownership
- View account status

### GST Details
- GST number
- Legal name
- Trade name
- State of registration
- Verification status

### Billing
- Billing contact
- Email preferences
- Invoice delivery settings

## API Endpoints
- `GET /organizations/:id` - Organization details
- `PATCH /organizations/:id` - Update organization
- `GET /organizations/:id/bank-accounts` - List bank accounts
- `POST /organizations/:id/bank-accounts` - Add bank account
- `GET /organizations/:id/gst-details` - GST details
