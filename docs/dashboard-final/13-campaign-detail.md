## Tabs

### Overview Tab
- Key metrics (enrollments, approval rate, payouts, avg order)
- Billing details with cost breakdown
- Campaign progress bars
- Enrollment status breakdown

### Enrollments Tab
- Table of all campaign enrollments
- Filters and bulk actions
- Links to individual enrollment review

### Statistics Tab
- Performance charts
- Trend analysis
- Time-based metrics

### Settings Tab
- Edit campaign details
- Manage deliverables
- Update terms

## API Endpoints
- `GET /campaigns/:id` - Campaign details
- `GET /campaigns/:id/stats` - Statistics
- `GET /campaigns/:id/billing` - Brand billing info
- `GET /campaigns/:id/performance` - Performance metrics
