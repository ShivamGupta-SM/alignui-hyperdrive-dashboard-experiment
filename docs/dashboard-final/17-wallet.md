## Data Sources

| Field | Source | Description |
|-------|--------|-------------|
| Available Balance | Blnk API | Real-time ledger balance |
| Held in Enrollments | Blnk API | Sum of active inflight holds |
| Credit Limit | `organization.creditLimit` | Admin-set overdraft limit |

## Transaction Types
- **Credit**: Funds added to wallet
- **Hold Created**: Funds reserved for enrollment
- **Hold Committed**: Funds finalized for payout
- **Hold Voided**: Funds released back to available

## API Endpoints
- `GET /organizations/:id/wallet` - Get wallet balances
- `GET /organizations/:id/transactions` - Transaction history
- `GET /organizations/:id/holds` - Active holds breakdown
