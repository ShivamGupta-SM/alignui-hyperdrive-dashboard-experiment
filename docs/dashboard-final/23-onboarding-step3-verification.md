## Verification Fields

### GST Number (Mandatory)
- 15-character GSTIN format
- Real-time verification via GST API
- Displays:
  - Legal name
  - Trade name
  - Registration status
  - Registered address
- Cannot proceed without valid GST

### PAN Number (Recommended)
- 10-character format
- Cross-verification with GST
- Optional but recommended for compliance

### CIN Number (Conditional)
- Required for Private Limited/LLP
- 21-character corporate identity number
- Format: UXXXXX[ST]YYYYPTCXXXXXX

## Verification Flow
1. User enters GST number
2. Click "Verify GST" button
3. System calls GST verification API
4. Success: Display verified details
5. Failure: Show error message

## API Endpoints
- `POST /verify/gst` - Verify GST number
- `POST /verify/pan` - Verify PAN number
- `PATCH /organizations/draft/:id` - Save verification data
