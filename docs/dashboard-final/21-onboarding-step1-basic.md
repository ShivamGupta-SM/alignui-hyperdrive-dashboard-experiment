## Fields

### Organization Name (Required)
- Unique identifier
- 3-100 characters
- Auto-generates slug (@acme-corporation)

### Description (Optional)
- Textarea, 500 char max
- Brief business overview

### Website (Optional)
- URL format validation
- Must start with http:// or https://

### Logo Upload (Optional)
- Drag-and-drop area
- Image formats: PNG, JPG, SVG
- Max size: 5MB
- Square aspect ratio recommended

## API Endpoints
- `POST /organizations/draft` - Save draft org data
- `POST /upload/logo` - Upload logo image
