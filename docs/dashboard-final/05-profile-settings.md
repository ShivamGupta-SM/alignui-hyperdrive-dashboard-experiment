## Features
- User avatar with initials
- Email and role display
- Organized sections: Account, Organization, Support
- Toggle switches for quick settings (Dark Mode)
- Arrow indicators for sub-panels
- Back button navigation in sub-panels
- Sign out action at bottom

## Menu Sections

### Account
- My Profile (edit personal info)
- Change Password
- Notification Preferences
- Dark Mode toggle

### Organization
- Organization Settings
- Team Member Management
- Billing & Payments

### Support
- Help Center
- Documentation
- Contact Support
