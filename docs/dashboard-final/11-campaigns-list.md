## Actions by Status

| Current Status | Available Actions |
|----------------|-------------------|
| Draft | Edit, Submit for Approval, Delete, Cancel |
| Pending Approval | View (waiting for admin) |
| Rejected | Edit, Resubmit for Approval, Cancel |
| Approved | Activate, Cancel |
| Active | Pause, End, View Stats |
| Paused | Resume, End, Cancel |
| Ended | Complete, Archive |
| Completed | Archive |
| Archived | Unarchive, Duplicate |

## API Endpoints
- `GET /campaigns` - List campaigns
- `GET /campaigns/:id/stats` - Campaign stats
- `POST /campaigns/:id/pause` - Pause campaign
- `POST /campaigns/:id/resume` - Resume campaign
- `POST /campaigns/:id/end` - End campaign
- `POST /campaigns/:id/archive` - Archive campaign
- `DELETE /campaigns/:id` - Delete draft campaign
